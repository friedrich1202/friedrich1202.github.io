What is Minecraft?

Minecraft is a sandbox game developed by Mojang Studios. In the game, players can interact with blocks in a 3D world. Features in the game include exploring the world, collecting resources, synthesizing items, and survival adventures.

Game features

  • Explore and build freely
  • Unknown dangers and adventures
  • Multiplayer online mode
  • Mods and custom map support
